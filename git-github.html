O que é o Git

O Git é uma ferramenta usada para controlar a versão do seu site ou app. O nome do app para usar o Git é Git Bash e para configura-lo é só seguir o tutorial:

1º primeiro confira se o git está instalado
git --version
2º agora confira os seus documentos
cd Documents;
3º abra o Documents
ls
4º agora configure
git config --global user.name (seu nome) git config --global user.email (seu e-mail)
5º crie uma pasta cd
mkdir *notas(ou outro nome)
ls{aqui mostra o que tem dentro da pasta}
cd *notas/
ls{aqui mostra o que tem dentro da pasta}
6º inicialize o git
git init
7º abra com o visual studio code
code .
8º escreva suas anotações, criando um arquivo com o nome de readme.md ,na pasta NOTAS.

O que é o Github

O Github é um site para mandar os seus arquivos, como se fosse um google drive só que maior e com a possibilidade que ver o perfil de outras pessoas, sem contar que caso perdemos o nosso e-mail podemos visualizar as coisas que postamos pois estamos em um site com nossas coisas.

Para configurar o Github é só se cadastrar nele, então coloque o seu e-mail pessoal e crie um senha

Agora para colocar o Git no Github é só seguir o tutorial:


Abra um terminal, no seu vscode,Verifique se existe chave ssh.


ls -al ~/.ssh


Adicione uma nova chave. (ID)


ssh-keygen -t ed25519 -C "your_email@example.com"
só fazer quando mudar de máquina.

Clique no "enter" até chegar em um quadrado com muitos caracteres.


Inicializar agente-ssh.eval


"$(ssh-agent -s)"

Adicionar chave ssh ao agente


ssh-add ~/.ssh/id_ed25519

Copiar chave ssh.


clip < ~/.ssh/id_ed25519.pub Adicionar chave no github Github -> Settings -> SSH and GPG keys -> NewSSH key -> Colar*Coloque um título que identifique a chave*
Testar conexãossh -T git@github.com pressione 'yes'

Caso você já tenha a chave é só seguir do copiar a chave em diante.

Build= um processo que serve para deixar o codigo de nosso app mais seguro (pois a maquina verá melhor do que o humano), e também deixa ele melhor para enviar para o público.

Uploud= é um processo de enviar o app para o público.

Backup= é um processo de atualizar os arquivos do app e salvar. Para caso alguém hackear os seus arquivos você ter eles salvos.
